# Starlight UI Tweaks

A Starlight plugin that extends and customizes core UI components:

- Replaces the theme toggle with an icon-based selector
- Adds navigation links to the navbar
- Enables ads in the table of contents
- Provides a customizable four-column footer

## Getting started

Please follow the documentation at [starlight-ui-tweaks.dlcastillop.com](https://starlight-ui-tweaks.dlcastillop.com/).
